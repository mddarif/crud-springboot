# 📚 Student Management System - Spring Boot CRUD REST API

## 📌 Project Overview
The **Student Management System** is a RESTful web application built using **Java Spring Boot and MySQL**.  
This application allows users to perform **CRUD operations (Create, Read, Update, Delete)** on student records using REST APIs.

The project demonstrates:
- Backend development with Spring Boot
- Database integration with MySQL
- REST API creation
- API testing using Postman

---

## 🛠 Technologies Used

- Java
- Spring Boot
- MySQL
- Spring Data JPA
- Maven
- REST API
- Postman

---

## 🚀 Features

✔ Add new student  
✔ View all students  
✔ Get student by ID  
✔ Update student information  
✔ Delete student
